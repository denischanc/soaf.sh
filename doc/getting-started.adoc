== Hello-world project

.Creation
[source, shell]
----
$> soaf.sh ACTION=create_prj PRJ_DIR=hello-world
$> cd hello-world
$> make all
----

.Test
[source, shell]
----
### Display usage
$> ./src/hello-world
### Action hello_world
$> ./src/hello-world ACTION=hello_world NAME=yourname
----

.Installation
[source, shell]
----
$> make install INSTALL_DIR=/usr/local/hello-world
$> /usr/local/hello-world/bin/hello-world
----
