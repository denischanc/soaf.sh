Install soaf.sh
===============

include::attributes[]

You must {url-home-soaf}/download.html[download^] a distribution :
*soaf.sh-x.y.z.tar.xz* for example. You want to install soaf.sh
in the directory : */usr/local/soaf.sh* for example.

.Installation
[source, shell]
----
$> tar xf soaf.sh-x.y.z.tar.xz
$> cd soaf.sh-x.y.z
$> make install INSTALL_DIR=/usr/local/soaf.sh
$> cd ..
$> rm -rf soaf.sh-x.y.z
$> export SOAF_HOME=/usr/local/soaf.sh
$> export PATH=$PATH:$SOAF_HOME/bin
$> soaf.sh
----
